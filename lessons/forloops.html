What is a For Loop?

A "for loop" does what it sounds like: it loops for a specified number of iterations. This can be useful for better automating your code. Upon each iteration of the for loop, the inner contents are run, though it is possible to manipulate each iteration. This per-iteration-manipulation will become more apparent when we see just how this loop works.




For Loop Syntax

Pictured is an example of a for loop delared in C++. There are three distinct parts to the for loop, each seperated by semicolons. The first declares a new variable "i" and assigns it a value of 1. The second declares the "for" part of the for loop. In this case, the loop will run so long as the variable "i" is less than 6. Also note that the number "6" can be replaced with a variable, making your code much more portable in the future. The last case declares what will happen to our variable "i" at the end of each iteration. In this example, at the end of each iteration "i" will be incremented by 1.

As for what happens during each iteration, we must look to what is included inside the subsequent "block" of code (encapsulated by curly brackets). In this case, we will output "i" and a space for every time this loop runs.

Giving us a final output of

1 2 3 4 5
