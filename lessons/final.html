How it Works

The goal of this mini game is to get the arrow from it's start position to the green tile. You can accomplish this by using the different C++ coding related buttons. Note that blue tiles represent a wall, red tiles represent an area your arrow can pass through, and green is the goal tile. If you choose a code button that results in your arrow crashing into the wall, you lose. To try again, simply refresh the page!
