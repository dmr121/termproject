Learn C++

An Interactive Guide




Learn the Fundamentals of C++ Programming

In this interactive website, you will be introduced to six different key concepts of the C++ programming language. Each Lesson will give a brief explenation on what the piece of C++ is, how it is used, and what it means. Each of these short mini-lessons will also include a small quiz for the user to test whether or not they fully understood the given lesson.


Topics Include:

  • Variables
  • Arrays
  • Conditionals
  • Switches
  • For Loops
  • While Loops

    • The Final Challenge

    After completing each of the lessons, a final challenge will be presented that represents some key concepts of C++ programming. While the game does not necessarily test your C++ programming capabilities, it does test whether or not you fully understand key programming concepts / logic.
